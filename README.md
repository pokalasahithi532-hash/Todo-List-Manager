# Todo-List-Manager
Description:
A simple console based To-Do List Manager is built using Java.
It allows user to add,view,delete,mark tasks as completed.
Tasks are stored in a file so that tasks remain after program exit.

Features:
->Add tasks
->View tasks with status(pending/complete)
->Delete tasks
->Mark tasks as completed
->Save tasks to file

Used Java,ArrayList,FileHandling.

File storage:
Tasks are stored in tasks.txt
